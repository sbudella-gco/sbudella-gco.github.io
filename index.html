Retrospective...

Disclosed vulnerabilities: [CVEs]

I've discovered vulnerabilities in security products and, more recently, in blockchain projects, focusing on cryptography, consensus protocols, and other issues like memory corruption bugs and race conditions.

Regular Positive Negative Inference Library for Scheme and Python:
[scheme README] [scheme src] [python src]

Simple and modular implementation of the RPNI algorithm as described in the book Grammatical Inference, Learning Automata and Grammars. Tested against the Stamina competition training and test suite. Available for Python and Scheme Lisp (requires Chicken Scheme).

Virtual Dynamic Shared Objects Support and Shared Page Subsystem for FreeBSD: [patch] [paper] [tgz]

The kernel modification aims to introduce a stable support for virtual dynamic shared objects to FreeBSD, by means of memory regions shared between the kernel and any user process.

Curuncula: [src]

Curuncula is a tool shipped as a loadable kernel module that detects virtualization rootkits that abuse the Intel debugging support facilities; rootkits that set the GD access flag are also detected.

Vsyscall Page Hijacking: [paper] [src]

Paper appeared on the Italian security magazine BFi, illustrating how overbloating features in the Linux kernel can be exploited to achieve syscall redirection in an inedited way, namely hybrid hooking between kernel space and user space.

Symbiotic Process Execution: [paper-italian]

Neither exec nor grugq's userland exec are necessary for code execution, since symbiotic sharing of process's resources is sufficient. Paper firstly appeared on the Italian security magazine BFi.

Claire, The Signal Context Fuzzer: [src]

The first fuzzer targeting the signal context of userland processes.

Reversed face recognition: [src]

Generation of human faces through linear combination of eigenfaces obtained from a training set. (meschach and libsdl needed. Best results achieved with The Japanese Female Facial Expression (JAFFE) Database).
